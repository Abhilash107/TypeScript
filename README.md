# About Ts:
- Ts is superset of js.(Layer over Js)
- ts = js + more precised way
- Type safety
    2 + "2" == '22' not in Ts
    null + 2 == 2 not in ts

- ts does static checking(compile timing like java and Go)
- Ts is converted to Js
- It's an development tool, the project still runs on Js.
